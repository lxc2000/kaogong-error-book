## 🎯 项目功能

这个小程序提供以下功能：

- **📝 添加错题** - 记录题目、答案、知识点、解析
- **🎲 随机抽题** - 从题库中随机抽取题目练习
- **📄 生成试卷** - 自定义题量，生成可打印试卷
- **📖 错题本** - 查看和管理错题
- **📊 统计功能** - 题量统计和知识点分布

## 💡 重要提示

### AppID 配置
当前使用的是测试号（用于开发测试）：
- 优点：无需注册，立即可用
- 限制：功能有限，无法正式发布

如果需要正式发布：
1. 访问 https://mp.weixin.qq.com/
2. 注册小程序
3. 获取正式 AppID
4. 在 `project.config.json` 中替换

### 图标资源
小程序底部导航栏需要图标：
- 位置：`miniprogram/images/`
- 尺寸：81×81 像素
- 数量：6 个（3 个功能 × 2 个状态）

**临时方案**：
如果暂时没有图标，可以在 `app.json` 中注释掉 `tabBar` 配置：
```json
// "tabBar": {...}  // 临时注释
```
