# 图标资源说明

## 📐 图标规格

**尺寸**：81 × 81 像素
**格式**：PNG（支持透明背景）
**色彩模式**：RGB

## 📋 需要的图标

### TabBar 图标（6个）

| 文件名 | 用途 | 颜色建议 |
|--------|------|----------|
| home.png | 首页（未选中） | 灰色 #999999 |
| home-active.png | 首页（选中） | 蓝色 #4A90D9 |
| add.png | 添加（未选中） | 灰色 #999999 |
| add-active.png | 添加（选中） | 蓝色 #4A90D9 |
| wrong.png | 错题本（未选中） | 灰色 #999999 |
| wrong-active.png | 错题本（选中） | 蓝色 #4A90D9 |

## 🎨 设计建议

1. **简洁明了**：图标应该直观表达功能
2. **统一风格**：所有图标保持一致的设计风格
3. **对比度**：确保选中状态足够明显
4. **线条粗细**：建议使用 2-3px 线条

## 🛠️ 获取图标的方式

### 方式 1: 使用在线图标库
- Iconfont：https://www.iconfont.cn/
- Flaticon：https://www.flaticon.com/
- IconPark：https://iconpark.oceanengine.com/

### 方式 2: 自定义绘制
- 使用 Figma、Sketch、Adobe XD 等设计工具
- 手工绘制简单图标

### 方式 3: 使用占位符（临时）
在开发初期，可以使用临时占位图标，后续替换。

## 📝 放置位置

将准备好的图标文件放在：
```
kaogong-error-book/miniprogram/images/
```

## ⚠️ 注意事项

- 文件名必须与 `app.json` 中配置的文件名完全一致
- 必须是 PNG 格式
- 尺寸必须为 81×81 像素
- 图标大小不应超过 40KB

## 🚀 快速开始

如果暂时没有图标，可以先注释掉 `app.json` 中的 `tabBar` 配置：

```json
{
  "pages": [...],
  "window": {...},
  // "tabBar": {...},  // 临时注释掉
  "style": "v2"
}
```

这样小程序仍可正常运行，只是没有底部导航栏。

---

**准备好图标后，将它们放入 images 目录即可！**
